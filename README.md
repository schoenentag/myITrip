## 프로젝트 소개
- IT 직무 관련업계 종사자에게 1:1 상담을 받고 동시에 부트캠프 수강 또한 가능하도록 하자는 취지에서 시작된 프로젝트
- 더 나아가 CBT 문제은행 기능을 추가해 문제를 풀고 사용자들끼리 의견과 정보를 공유할 수 있는 사이트

## 개발 기간
2022.09.26 ~ 2022.10.19
